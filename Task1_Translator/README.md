## Universal Language Translator

Live Demo : https://monisha-2707.github.io/codealpha_tasks/Task1_Translator/

**Features**
- Automatic language detection
- Translation between multiple languages
- Romanized ENglish typing support
- speech functionality
- copy translated text instantly
- support for Indian and international languages

**Technologies used:**

HTML,CSS,JavaScript,Web Speech API,Google Translation APIs

**How to Run:**
1. Launch `index.html` in a web browser.
2. Enter text the text to be translated.
3. (Optional) Enable the **Live translation** checkbox for automatic translation while       typing.
4. The input language will be automatically detected.
5. Select the target language you want the text translated into.
6. View the translated output.
7. Click the **Speak** button to hear the translated text.

**Browser compatibility:**

THis project is best experienced in modern browsers such as google chrome and microsoft edge.

**Note:**
1. Speech Recognition and Text to speech rely on browser and operating system support.
2. Some languages may not have native voice support on all devices.
3. If a voice featur does not work in one browser, try Chrome or Edge.
