## Browser Compatibility

This project is best experienced in modern browsers such as Google Chrome and Microsoft Edge.

Note:
- Speech Recognition and Text-to-Speech rely on browser and operating system support.
- Voice availability may vary depending on the installed language packs and browser.
- If a voice feature does not work in one browser, try Chrome or Edge.

A multilingual language translator developed as part of the CodeAlpha Internship Program.

## Features
- Automatic language detection
- Translation between multiple languages
- Romanized English typing support
- Live translation mode
- Text-to-speech functionality
- Copy translated text instantly
- Support for Indian and international languages

## Technologies Used
- HTML
- CSS
- JavaScript
- Web Speech API

## How to Run
1. Open the project folder.
2. Launch `task1.html` in a web browser.
3. For the best voice support, use Google Chrome.
4. Enter text, select languages, and translate instantly.
