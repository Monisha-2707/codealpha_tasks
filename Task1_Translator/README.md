# Universal Language Translator

A multilingual language translator developed as part of the CodeAlpha Internship Program.

## Features
- Automatic language detection
- Translation between multiple languages
- Romanized English typing support
- Live translation mode
- Text-to-speech functionality
- Copy translated text instantly
- Support for Indian and international languages

## Technologies Used
- HTML
- CSS
- JavaScript
- Web Speech API

## How to Run
1. Open the project folder.
2. Launch `task1.html` in a web browser.
3. For the best voice support, use Google Chrome.
4. Enter text, select languages, and translate instantly.
