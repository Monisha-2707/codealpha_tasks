# Hospital FAQ Chatbot

**Live Demo:** 

https://monisha-2707.github.io/codealpha_tasks/Task2_Hospital_FAQ_Chatbot/

**Features**
- Interactive chatbot interface
- Hospital-related FAQs
- Smart response matching
- Confidence score display
- Dark mode support
- Quick suggestion chips


**Technologies Used:**   

1. HTML
2. CSS
3. JavaScript


**How to Run:**

1. Launch `index.html` in a web browser.
2. Type a hospital-related question in the chat input box.
3. Press **send** or hit Enter to submit your query.
4. The chatbot will analyze the question and search for the most relevant response.
5. view the chatbot's answer along with the confidence score.
6. use the suggestion chips for quick access to common hospital-related queries.
7. Toggle between Light and dark mode for a better user experience.
